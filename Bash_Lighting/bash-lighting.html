Variable mangling
${VAR#*.} short beginning
${VAR##*.} long beginning
${VAR%.*} short end
${VAR%%.*} long end
${VAR:start:length} cut characters out

Example: 

 VAR="filename.and.extension"
 echo "VAR# ${VAR#*.}"
 echo "VAR## ${VAR##*.}"
 echo "VAR% ${VAR%.*}"
 echo "VAR%% ${VAR%%.*}"
 NAME=`echo ${VAR} | cut -d. -f1`
 echo "VAR%% ${VAR##${NAME}.}"
 echo "VAR: ${VAR:9:3}"

Command Line Editing

Example: 

 Emacs Flavor ( set -o emacs )
   ^a - beginning of line
   ^e - end of line
   ^f - forward a character
   ^b - backward a character
 Vi Flavor ( set -o vi )
   0 - beginning of line
   $ - end of line
   i - insert
   x - delete character
