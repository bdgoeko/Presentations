Command Line Editing

Example: 

 Emacs Flavor ( set -o emacs )
   ^a - beginning of line
   ^e - end of line
   ^f - forward a character
   ^b - backward a character
 Vi Flavor ( set -o vi )
   0 - beginning of line
   $ - end of line
   i - insert
   x - delete character

Command History

Example: 

Emacs Flavor ( set -o emacs )
  ^p - previous Command
  ^n - next command
  ^r - search backwards
Vi Flavor ( set -o vi )
  j - previous command
  k - next command
